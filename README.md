# Scanner
Python scanner built upon the scapy library.

## setup
```bash
apt-get install python3
sudo apt-get install python3-venv
pip install scapy
```
## setup with venv
```bash
apt-get install python3
sudo apt-get install python3-venv
python3 -m venv venv
source ./venv/bin/activate
pip install scapy
```

installing Scapy can be useful for testing / debugging

sudo apt intall scapy
